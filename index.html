My Projects

Learn Circassian - Card Game: is a personal project designed as a fun and interactive method to learn Circassian vocabulary using memory cards.
The goal is to revitalize the Circassian language, which faces the risk of extinction,
by making learning engaging and accessible.
Players match images (Red Cards) with corresponding Circassian words (Blue Cards) to reinforce language comprehension and retention.
NoteShare: NoteShare is a robust note sharing platform built on the Spring Boot MVC pattern,
leveraging RESTful APIs and Thymeleaf for dynamic web content.
It enables users to effortlessly create, share, and manage notes, fostering collaborative learning and knowledge sharing among its community members.
Maze 2D a JavaFX-based game project where players navigate through a maze. It utilizes threading to manage concurrent tasks efficiently.
